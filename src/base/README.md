
# Base

A devcontainer feature

## Example Usage

```json
"features": {
    "ghcr.io/ridhwaans/devcontainer-features/base:1": {
        "version": "latest"
    }
}
```

## Options

| Options Id | Description | Type | Default Value |
|-----|-----|-----|-----|
| username | Username | string | ridhwaans |
| setTheme | Set theme for vim and zsh | boolean | true |
